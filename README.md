## Paytm Insider Assignment

## Install
Install all dependencies.
```
npm install
```

## How to start
Start local server
```
npm run server
```

Start app
```
npm start
```

## Structure
> Web site created using `create-react-app`.

```
/server
|  |- server.js
/src
|- /common
|      |- /components
|      |- api.js
|- /components
|- /Styles
|      |- _global.scss
|      |- _mixin.scss
|      |- _reset.scss
|      |- _variables.scss
|- /utils
|      |- utils.js
|- App.js
|- App.scss
|- index.js

```

> All images are stored on cloud image hosting site `cloudinary`.
